# Snake Game

A game of snake, written in Hack using [`Nuxed`](https://github.com/nuxed)

## Local Setup

### Installation

```sh
git clone git@github.com:azjezz/snake-game.git snake-game
cd snake-game
composer install
composer dump-autoload
```

### Start playing

```sh
hhvm bin/snake.hack play
```
